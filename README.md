# Flight Compensation Checker

Free, instant flight compensation eligibility checker covering EU261, UK261, APPR, Montreal Convention, ANAC 400/2022, and Israel Aviation Services Law.

Live site: [compensation.printmoneylab.com](https://compensation.printmoneylab.com)

## About

A non-developer's experiment in building AI-first functional tools — web tools rebuilt for the era when AI agents and chatbots are the primary traffic source.

Part of [printmoneylab.com](https://printmoneylab.com) — documenting the journey of building automated income streams with AI.

## Stack

- [Astro](https://astro.build/) — static site generator
- [Cloudflare Pages](https://pages.cloudflare.com/) — hosting & deployment
- GitHub → Cloudflare Pages auto-deploy

## Regulations Covered

- **EU261** — €250–600 for flights in/out of EU
- **UK261** — £220–520 for flights in/out of UK  
- **APPR (Canada)** — CAD $125–1,000
- **Montreal Convention** — up to ~$1,700 for baggage delays
- **ANAC 400/2022 (Brazil)**
- **Israel Aviation Services Law** — ILS 1,320–3,180
